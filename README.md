# Malicious Domains Threat Intelligence Feeds

This repository contains a collection of **publicly available threat intelligence domain feeds** used for:
- Web, email, and DNS filtering
- SOC enrichment & investigation
- Threat hunting & detection content validation
- OSINT research
- Blocklist generation for defensive security

All domain sources are stored inside the `sources/` directory.
